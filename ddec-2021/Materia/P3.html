Tipografías en CSS

Una de las características básicas del CSS es que permite aplicar tipografías y pesos a los textos. Para implementar una tipografía, se utiliza la propiedad font-family en CSS: este atributo toma uno o más nombres de familia de tipos de letra y el navegador recorre esta lista hasta que encuentra un tipo de letra disponible en el sistema en el que se ejecuta; sin embargo, la fuente tipográfica debe estar instalada en el dispositivo del usuario, de lo contrario al abrir el archivo html se mostrará la fuente por defecto del navegador. Para solucionar esto se pueden utilizar tipografías web, las que deben declararse en el head del archivo HTML para poder vincularse correctamente y luego dentro de la hoja de estilos modificar los pesos seleccionados a cada título o cuerpo de texto. El uso de tipografías web permite que cualquier navegador que las admita pueda visualizarlas.
