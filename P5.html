¿Qué es el grid?

El grid layout es un sistema bidimensional, que transforma un elemento HTML en una cuadricula, el cual permite distribuir el contenido en filas y columnas, y tiene muchas características que facilitan la creación de diseños complejos. Los elementos hijos de este pueden ser posicionados dentro de las celdas de esta cuadricula, o en áreas definidas arbitrariamente con reglas CSS. Para activar la cuadrícula grid hay que utilizar sobre el elemento contenedor la propiedad display y especificar el valor grid o inline-grid. Establece una cuadrícula con ítems en línea, de forma equivalente a inline-block. Establece una cuadrícula con ítems en bloque, de forma equivalente a block.
